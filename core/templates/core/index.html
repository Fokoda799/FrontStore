🛒 Store API

Welcome to the Store API. This API provides endpoints to manage products, collections, carts, and orders.

Available Endpoints

  • /store/products/ — List all products
  • /store/collections/ — List product collections
  • /store/carts/ — Create and manage carts
  • /store/orders/ — Manage orders
  • /play/hello/ — Test endpoint

Example Query

Filter products by collection:

/store/products/?collection_id=1

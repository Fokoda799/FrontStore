Example Queries
HTTP Request Examples
# Filter products by collection
GET /store/products/?collection_id=1

# Filter products by price range
GET /store/products/?price__gt=10&price__lt=100

# Search products
GET /store/products/?search=font&ordering=-price

# Authenticate and get JWT
POST /auth/jwt/create/
  "username": "your_username",
  "password": "your_password"

# Add item to cart
POST /store/carts/{cart_pk}/items/
  "product_id": 42,
  "quantity": 2
Authentication
🔑
JWT Bearer Authentication
Most endpoints require authentication. Obtain a token pair from /auth/jwt/create/, then include the access token in every request header.
Authorization: JWT <your_access_token>
